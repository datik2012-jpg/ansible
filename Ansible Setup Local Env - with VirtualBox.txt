How to do SSJ connection not ask for the fingerprint and do in on all servers:


Full bootstrap playbook (production‑ready)
This playbook runs on localhost (your controller) and:

scans all hosts

builds a known_hosts file

installs it

verifies connectivity

yaml
---
- name: Bootstrap SSH trust for all inventory hosts
  hosts: localhost
  gather_facts: no

  vars:
    inventory_hosts: "{{ groups['all'] }}"

  tasks:

    - name: Gather SSH host keys
      command: "ssh-keyscan -H {{ item }}"
      loop: "{{ inventory_hosts }}"
      register: scanned_keys

    - name: Write collected keys to temporary file
      copy:
        dest: /tmp/known_hosts.generated
        content: "{{ scanned_keys.results | map(attribute='stdout') | join('\n') }}"

    - name: Install known_hosts file system-wide
      become: yes
      copy:
        src: /tmp/known_hosts.generated
        dest: /etc/ssh/ssh_known_hosts
        mode: '0644'

    - name: Verify connectivity
      command: "ansible all -m ping -i inventory"
      register: ping_output

    - debug:
        var: ping_output.stdout
🧠 Why this is production‑safe
No manual SSH acceptance

Host key checking stays enabled (important for security)

Centralized trust file (/etc/ssh/ssh_known_hosts)

Auditable onboarding

Repeatable automation
